## P0-31: 크롤링 가능성 + 법적 검토

### 한국 웹 크롤링 법적 기준 (대법원 2022도1533 참조)

| 기준 | 합법 | 위법 |
|------|------|------|
| robots.txt | 허용 범위 내 수집 | 차단 영역 수집 |
| 이용약관 | 약관에 금지 조항 없음 | "자동 수집 금지" 명시 위반 |
| 사용 목적 | 정보 참조, 검색 색인 | **상업적 복제/판매** |
| 서버 부하 | 합리적 속도 (rate limit) | 과도한 요청으로 서버 장애 |

### 사이트별 평가

| 사이트 | 크롤링 가능성 | 위험도 | 비고 |
|--------|-------------|--------|------|
| 올리브영 | 낮음 | **높** | 이용약관에 자동 수집 금지 가능성. 상업적 데이터 활용은 법적 리스크 |
| 네이버 플레이스 | 낮음 | **높** | robots.txt 제한적. 네이버가 적극적으로 크롤링 차단 |
| 화해/글로우픽 | 매우 낮음 | **높** | 이용약관 명시적 금지 |
| 브랜드 공식 사이트 | 중 | **낮** | 공개 제품 정보는 상대적으로 안전. 이미지는 주의 |

### 권장

## P0-30: 외부 API 필드 매핑

### 카카오 로컬 API (권장 1순위)

```
GET /v2/local/search/keyword.json?query=강남+올리브영

반환 필드:
- place_name: 매장명 → stores.name
- address_name: 지번 주소 → stores.address
- road_address_name: 도로명 주소 → stores.address
- x, y: 경도, 위도 → stores.location (GEOGRAPHY)
- category_name: "가정,생활 > 화장품" → stores.store_type
- phone: 전화번호
- place_url: 카카오맵 링크 → stores.external_links

누락 필드: 영업시간, 영어 지원, 관광객 서비스 → 수동 보완 필요
일일 한도: 30만건 (MVP 충분)
```

### 네이버 검색 API (보조)

```
GET /v1/search/local.json?query=강남+뷰티+클리닉

반환 필드:
- title: 장소명
- address, roadAddress: 주소
- mapx, mapy: 좌표
- category: 카테고리
- link: 네이버 플레이스 링크

누락 필드: 영업시간, 상세 정보 → 수동 보완 필요
일일 한도: 25,000건
```

### Google Places API (영어 데이터 보강)

```
필드 티어별 가격:
- Essentials ($0~): 좌표, 주소
- Pro ($): 이름, 아이콘, 사진 ID
- Enterprise ($$): 전화, 웹사이트, 영업시간, 평점
- Enterprise+Atmosphere ($$$): 리뷰, 편의시설

월 $200 무료 크레딧으로 ~10,000건 Enterprise 조회 가능
한국 데이터: 관광지/대형매장은 양호, 소규모 클리닉은 부족
```
