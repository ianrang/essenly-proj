# 아데노신 (Adenosine)

> 이 정보는 일반 참고용이며 의료 조언이 아닙니다. 피부 상태에 따라 전문의와 상담하세요.

## 기본 정보
- INCI: Adenosine
- 별명: 아데노씬
- 기능: 주름 개선(한국 식약처 고시 주름 개선 기능성 원료), 콜라겐·엘라스틴 합성 촉진, 상처 치유, 항염
- K-뷰티에서: 한국 식약처 인정 주름 개선 기능성 성분으로, K-뷰티 안티에이징 제품에 필수적으로 포함. 설화수, 후 등 한방 브랜드부터 이니스프리, 미샤까지 폭넓게 사용. 기능성 화장품 표기 시 자주 등장

## 피부타입별 적합도
- 건성(dry): 적합 — 자극 없이 안티에이징 효과를 제공하며 보습 제품과 잘 어울림
- 지성(oily): 적합 — 성분 자체가 유분이 없어 피부 타입에 관계없이 사용 가능
- 복합성(combination): 적합 — 가벼운 제형으로 전체 얼굴에 사용 가능
- 민감성(sensitive): 적합 — 자극이 매우 낮아 민감성 피부에서도 안전하게 사용 가능
- 보통(normal): 적합 — 예방적 안티에이징으로 20대 후반부터 사용 추천

## 시너지 성분
- 레티놀: 레티놀의 콜라겐 생성 촉진 + 아데노신의 안티에이징으로 주름 개선 시너지
- 펩타이드: 여러 경로로 콜라겐을 자극하여 복합적 안티에이징 효과

## 주의사항
- 매우 안전한 성분으로 특별한 주의사항이 적음
- 효과를 체감하려면 꾸준한 사용이 필요 (즉각적 효과보다는 장기적 개선)
- 기능성 화장품 기준 0.04% 이상 함유 시 주름 개선 효과 인정
