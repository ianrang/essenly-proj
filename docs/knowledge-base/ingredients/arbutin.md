# 알부틴 (Arbutin)

> 이 정보는 일반 참고용이며 의료 조언이 아닙니다. 피부 상태에 따라 전문의와 상담하세요.

## 기본 정보
- INCI: Alpha-Arbutin / Arbutin
- 별명: 알파알부틴, 베타알부틴
- 기능: 미백(티로시나제 활성 억제로 멜라닌 생성 감소), 색소침착 완화, 기미·잡티 개선
- K-뷰티에서: 한국 식약처 고시 미백 기능성 원료. 투쿨포스쿨, 이니스프리, 스킨1004 등에서 알파알부틴 세럼으로 출시. 하이드로퀴논의 안전한 대안으로 K-뷰티 미백 라인에 필수 성분

## 피부타입별 적합도
- 건성(dry): 적합 — 자극이 적어 건조 피부에서도 안전하게 미백 관리 가능
- 지성(oily): 적합 — 여드름 후 색소침착(PIH) 개선에 효과적
- 복합성(combination): 적합 — 전체 피부 톤 균일화에 도움
- 민감성(sensitive): 적합 — 하이드로퀴논보다 훨씬 순하여 민감 피부에도 사용 가능
- 보통(normal): 적합 — 예방적 미백 관리와 톤 개선에 효과적

## 시너지 성분
- 나이아신아마이드: 서로 다른 미백 메커니즘으로 시너지 (알부틴=생성 억제, 나이아신아마이드=전달 억제)
- 비타민C: 항산화 + 미백 이중 효과로 색소침착 개선 가속

## 주의사항
- 알파알부틴이 베타알부틴보다 안정적이고 효과적 (알파알부틴 제품 권장)
- 미백 효과는 꾸준한 사용(최소 8주)이 필요
- 자외선 차단제 병행 필수 — 미백 성분 사용 시 자외선 노출로 색소침착이 악화될 수 있음
- 매우 안전한 성분이나 고농도 사용 시 드물게 자극 가능
