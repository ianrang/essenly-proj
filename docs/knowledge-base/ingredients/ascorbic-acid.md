# 아스코르브산 (Ascorbic Acid)

> 이 정보는 일반 참고용이며 의료 조언이 아닙니다. 피부 상태에 따라 전문의와 상담하세요.

## 기본 정보
- INCI: Ascorbic Acid
- 별명: 비타민 C, L-아스코르브산
- 기능: 항산화(활성산소 중화), 콜라겐 합성 촉진, 미백(티로시나제 억제), 자외선 손상 방어, 피부 톤 균일화
- K-뷰티에서: 클레어스 비타민 드롭, 바이 위시트렌드 순수 비타민C 세럼 등이 인기. K-뷰티에서는 안정화 비타민C 유도체(에틸아스코르빌에테르, 아스코르빌글루코사이드)를 활용한 순한 제형도 많음

## 피부타입별 적합도
- 건성(dry): 보통 — 콜라겐 합성으로 탄력 개선에 도움. 단, 고농도(15% 이상)는 건조감 유발 가능
- 지성(oily): 적합 — 항산화와 미백 효과. 피지 산화를 방지하여 블랙헤드 예방
- 복합성(combination): 적합 — 전체적인 피부 톤 개선에 효과적
- 민감성(sensitive): 주의 — 순수 L-아스코르브산은 pH가 낮아(2.5~3.5) 따가울 수 있음. 비타민C 유도체 제품 권장
- 보통(normal): 적합 — 일상적 항산화·미백 관리에 가장 효과적인 성분 중 하나

## 시너지 성분
- 비타민 E(토코페롤): 함께 사용 시 항산화 효과가 4배 증가한다는 연구 결과
- 페룰산(Ferulic Acid): 비타민 C+E+페룰산 조합은 자외선 방어 시너지가 매우 우수

## 주의사항
- 산화에 매우 취약하여 빛·열·공기에 노출되면 갈변(효과 감소). 어둡고 서늘한 곳에 보관
- 개봉 후 2~3개월 내 사용 권장
- 나이아신아마이드와 동시 도포 시 일부 사용자에서 홍조 보고. 시간 간격을 두거나 아침/저녁 분리 권장
- 아침 사용 시 반드시 자외선 차단제 병행 (비타민C 자체가 광감작은 아니나 항산화 보호를 위해)
