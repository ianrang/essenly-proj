# 아젤라인산 (Azelaic Acid)

> 이 정보는 일반 참고용이며 의료 조언이 아닙니다. 피부 상태에 따라 전문의와 상담하세요.

## 기본 정보
- INCI: Azelaic Acid
- 별명: 아젤라산, 노난디오산
- 기능: 항균(여드름균 억제), 항염, 미백(비정상 멜라노사이트 선택적 억제), 각질 정상화, 주사(Rosacea) 완화
- K-뷰티에서: 코스알엑스 등에서 아젤라인산 함유 세럼 출시. 한국에서는 10% 이하 화장품 원료로 사용 가능. 피부과에서는 15~20% 농도의 전문 제품도 처방

## 피부타입별 적합도
- 건성(dry): 보통 — 미백·항염 효과는 좋으나 약간의 건조감이 있을 수 있어 보습 병행 필요
- 지성(oily): 적합 — 여드름 관리와 피지 조절에 효과적이며 모공 내 각질 정상화
- 복합성(combination): 적합 — 여드름 부위에 부분 사용하거나 전체 사용 모두 가능
- 민감성(sensitive): 적합 — 주사(Rosacea) 치료에 사용될 만큼 민감 피부에 적합. 초기 따가움은 적응 후 감소
- 보통(normal): 적합 — 여드름 예방과 피부 톤 균일화에 효과적

## 시너지 성분
- 나이아신아마이드: 항염 + 미백 시너지. 여드름 후 색소침착 관리에 효과적 조합
- 레티놀: 각질 정상화와 세포 턴오버 촉진으로 여드름·안티에이징 동시 관리

## 주의사항
- 사용 초기 따가움, 가벼운 작열감이 있을 수 있으나 대부분 1~2주 내 적응
- 비타민C와 동시 사용 가능 (pH 범위가 유사하여 호환)
- 임신 중에도 비교적 안전하다고 알려져 있으나 전문의 상담 권장
- 효과 체감까지 최소 4~8주 꾸준한 사용 필요
