# 병풀추출물 (Centella Asiatica Extract)

> 이 정보는 일반 참고용이며 의료 조언이 아닙니다. 피부 상태에 따라 전문의와 상담하세요.

## 기본 정보
- INCI: Centella Asiatica Extract
- 별명: 시카(CICA), 센텔라, 호랑이풀, 마데카소사이드
- 기능: 피부 진정, 상처 치유 촉진, 콜라겐 합성 촉진, 항염, 피부 장벽 회복
- K-뷰티에서: K-뷰티 시카 열풍의 주역. 닥터자르트 시카페어 크림, COSRX 센텔라 워터 알코올프리 토너, 아이소이 시카 크림 등이 대표적. 시카 성분은 크림, 세럼, 선크림까지 다양한 카테고리에 활용

## 피부타입별 적합도
- 건성(dry): 적합 — 장벽 회복과 보습 지원으로 건조한 피부 진정
- 지성(oily): 적합 — 가벼운 텍스처의 시카 제품이 많아 유분 부담 없이 진정 효과
- 복합성(combination): 적합 — 자극 부위 진정과 전체적인 피부 컨디션 개선
- 민감성(sensitive): 적합 — 민감성 피부를 위한 대표 진정 성분. 아토피 피부에도 사용 가능
- 보통(normal): 적합 — 외부 자극(자외선, 미세먼지) 후 피부 회복에 효과적

## 시너지 성분
- 마데카소사이드: 센텔라의 핵심 활성 성분으로, 농축된 형태가 더 강력한 재생 효과
- 판테놀: 진정 + 보습 시너지로 손상된 피부 회복에 탁월

## 주의사항
- 국화과 알레르기가 있는 경우 드물게 교차 반응 가능
- 센텔라 추출물의 품질은 활성 성분(아시아티코사이드, 마데카소사이드) 함량에 따라 차이가 큼
- 일반적으로 자극이 매우 낮으나, 첫 사용 시 패치 테스트 권장
