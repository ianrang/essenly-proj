# 세라마이드NP (Ceramide NP)

> 이 정보는 일반 참고용이며 의료 조언이 아닙니다. 피부 상태에 따라 전문의와 상담하세요.

## 기본 정보
- INCI: Ceramide NP
- 별명: 세라마이드 3, 피토세라마이드
- 기능: 피부 장벽 구성 성분으로서 장벽 복구, 수분 손실 방지(TEWL 감소), 보습 유지
- K-뷰티에서: 아토팜 MLE 크림, 일리윤 세라마이드 아토 크림 등 장벽 강화 제품에 핵심 성분. 한국의 세라마이드 기술력은 세계적으로 인정받으며, MLE(Multi-Lamellar Emulsion) 기술을 적용한 제품이 특징

## 피부타입별 적합도
- 건성(dry): 적합 — 피부 장벽의 핵심 지질 성분으로 건조 피부의 근본 원인 해결
- 지성(oily): 보통 — 장벽 강화에 도움이 되나 크림 베이스가 무거울 수 있음. 가벼운 로션 제형 선택
- 복합성(combination): 적합 — 유수분 밸런스를 맞추는 데 도움
- 민감성(sensitive): 적합 — 손상된 장벽 회복에 필수적. 아토피 피부에 효과적이라는 다수 연구 존재
- 보통(normal): 적합 — 예방적 장벽 관리와 계절 변화 시 보호

## 시너지 성분
- 콜레스테롤 + 지방산: 세라마이드:콜레스테롤:지방산 = 1:1:1 비율이 피부 장벽과 가장 유사
- 히알루론산: 내부 수분 공급(HA) + 외부 수분 유지(세라마이드) 조합

## 주의사항
- 세라마이드 단독보다 콜레스테롤, 지방산과 함께 배합된 제품이 더 효과적
- 합성 세라마이드와 식물성 세라마이드 모두 효과가 확인됨
- 자극이 거의 없어 특별한 주의사항은 적으나, 제품 내 다른 성분에 의한 자극 가능성은 확인 필요
