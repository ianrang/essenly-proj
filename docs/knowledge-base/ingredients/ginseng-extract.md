# 인삼추출물 (Ginseng Extract)

> 이 정보는 일반 참고용이며 의료 조언이 아닙니다. 피부 상태에 따라 전문의와 상담하세요.

## 기본 정보
- INCI: Panax Ginseng Root Extract
- 별명: 홍삼, 산삼, 진세노사이드
- 기능: 항산화(진세노사이드), 혈행 촉진, 콜라겐 합성 지원, 피부 탄력 개선, 안티에이징
- K-뷰티에서: 한국 대표 한방 원료. 설화수 자음생 라인, 더후 비첩자생 에센스 등 프리미엄 한방 화장품의 핵심 성분. 홍삼(증숙 인삼)은 진세노사이드 함량이 높아 더 강력한 효과. K-뷰티 럭셔리 세그먼트의 상징

## 피부타입별 적합도
- 건성(dry): 적합 — 영양 공급과 보습력이 뛰어나 건조하고 푸석한 피부에 윤기 부여
- 지성(oily): 보통 — 영양감이 풍부하여 무거울 수 있으나, 가벼운 에센스 제형은 사용 가능
- 복합성(combination): 보통 — 건조 부위 위주로 사용하거나 가벼운 제형 선택
- 민감성(sensitive): 보통 — 일반적으로 자극이 낮으나 인삼 알레르기 가능성 확인 필요
- 보통(normal): 적합 — 안티에이징과 피부 활력 증진에 효과적

## 시너지 성분
- 꿀/프로폴리스: 한방 영양 성분 + 천연 보습으로 고영양 안티에이징 케어
- 아데노신: 콜라겐 합성을 여러 경로로 촉진하여 주름 개선 시너지

## 주의사항
- 인삼 알레르기가 있는 경우 사용 금지
- 인삼 추출물의 품질은 재배 연수, 가공법(백삼/홍삼/흑삼)에 따라 크게 차이
- 향이 강한 편이므로 향에 민감한 사용자는 제품 선택 시 확인
- 임신 중 인삼 성분 사용에 대해서는 전문의 상담 권장
