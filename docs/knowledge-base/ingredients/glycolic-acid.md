# 글리콜산 (Glycolic Acid)

> 이 정보는 일반 참고용이며 의료 조언이 아닙니다. 피부 상태에 따라 전문의와 상담하세요.

## 기본 정보
- INCI: Glycolic Acid
- 별명: AHA (Alpha Hydroxy Acid), 알파하이드록시산, 글라이콜산
- 기능: 각질 제거(표피 세포 간 결합 약화), 피부결 개선, 콜라겐 생성 촉진, 색소침착 완화, 모공 관리
- K-뷰티에서: COSRX AHA 7 화이트헤드 파워 리퀴드, 이즈앤트리 AHA 에센스 등이 대표적. K-뷰티에서는 저농도(5~8%) 매일 사용 방식이 인기이며, 필링 패드 형태도 많이 출시

## 피부타입별 적합도
- 건성(dry): 보통 — 각질 제거로 후속 보습제 흡수를 높이나, 과도한 사용 시 건조 악화. 주 1~2회 권장
- 지성(oily): 적합 — 각질과 피지 제거로 모공 관리. 화이트헤드 개선에 효과적
- 복합성(combination): 적합 — T존 위주로 사용하거나 낮은 농도로 전체 사용
- 민감성(sensitive): 주의 — AHA 중 분자가 가장 작아 침투력이 높고 자극이 강한 편. 만델릭산(AHA) 대안 고려
- 보통(normal): 적합 — 주 2~3회 사용으로 꾸준한 각질 관리와 피부결 개선

## 시너지 성분
- 히알루론산: 각질 제거 후 수분 공급으로 건조감 보완과 촉촉한 피부결
- 나이아신아마이드: 각질 제거 + 장벽 강화로 자극을 최소화하면서 효과 극대화

## 주의사항
- 자외선 민감도를 높이므로 반드시 자외선 차단제 병행 필수
- 레티놀, BHA와 동시 사용 시 과도한 자극. 교대 사용 권장
- 처음 사용 시 5% 이하 농도로 시작하여 피부 적응 확인
- pH 3~4에서 효과가 가장 좋으나, 낮은 pH는 자극을 유발할 수 있음
