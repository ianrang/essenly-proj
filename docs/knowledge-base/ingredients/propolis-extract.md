# 프로폴리스추출물 (Propolis Extract)

> 이 정보는 일반 참고용이며 의료 조언이 아닙니다. 피부 상태에 따라 전문의와 상담하세요.

## 기본 정보
- INCI: Propolis Extract
- 별명: 봉교, 벌 프로폴리스
- 기능: 항균, 항염, 항산화, 보습, 피부 진정, 상처 치유 촉진
- K-뷰티에서: COSRX 풀핏 프로폴리스 라인, 토니모리 프로폴리스 타워 배리어 크림, 스킨푸드 로얄허니 프로폴리스 에센스 등이 대표적. K-뷰티에서 꿀·프로폴리스 성분은 영양 공급과 윤기 부여 콘셉트로 인기

## 피부타입별 적합도
- 건성(dry): 적합 — 풍부한 영양 성분과 보습력으로 건조 피부에 윤기 부여
- 지성(oily): 보통 — 항균·항염 효과로 여드름 관리에 도움이나, 일부 제품은 무거울 수 있음
- 복합성(combination): 적합 — 진정과 영양 공급의 균형 잡힌 케어
- 민감성(sensitive): 보통 — 항염 효과가 있으나, 프로폴리스 알레르기가 있으면 사용 금지
- 보통(normal): 적합 — 전반적인 피부 건강과 윤기 개선에 효과적

## 시너지 성분
- 꿀(Honey): 프로폴리스+꿀 조합으로 보습과 영양 공급 극대화
- 나이아신아마이드: 항염 + 장벽 강화로 트러블 피부 종합 케어

## 주의사항
- 벌 관련 제품 알레르기가 있는 경우 반드시 패치 테스트 후 사용
- 꿀벌 알레르기, 화분 알레르기가 있는 사람은 교차 반응 위험
- 비건 사용자는 동물 유래 성분임을 유의
- 프로폴리스 함량과 산지에 따라 품질 차이가 큼
