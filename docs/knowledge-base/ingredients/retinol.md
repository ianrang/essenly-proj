# 레티놀 (Retinol)

> 이 정보는 일반 참고용이며 의료 조언이 아닙니다. 피부 상태에 따라 전문의와 상담하세요.

## 기본 정보
- INCI: Retinol
- 별명: 비타민 A, 레틴올
- 기능: 콜라겐 생성 촉진, 세포 턴오버 가속, 주름 개선, 색소침착 완화, 여드름 관리
- K-뷰티에서: 이니스프리 레티놀 시카 세럼, 코스알엑스 리얼핏 레티놀 세럼 등 안정화 레티놀 제형이 인기. 캡슐화·리포좀 기술로 자극을 줄인 K-뷰티 특유의 순한 레티놀 제품이 많음

## 피부타입별 적합도
- 건성(dry): 주의 — 초기 건조감과 각질이 발생할 수 있으므로 저농도(0.025%)부터 시작하고 보습을 충분히
- 지성(oily): 적합 — 피지 조절과 모공 관리에 효과적. 비교적 자극을 잘 견딤
- 복합성(combination): 보통 — 건조 부위에 주의하며 사용. 부분 도포 가능
- 민감성(sensitive): 주의 — 레티놀 반응(홍조, 각질, 따가움)이 강하게 나타날 수 있음. 바쿠치올 등 대안 성분 고려
- 보통(normal): 적합 — 안티에이징 목적으로 효과적. 점진적 농도 증가 권장

## 시너지 성분
- 히알루론산: 레티놀 사용 시 발생하는 건조감을 보완하여 자극 최소화
- 나이아신아마이드: 레티놀의 자극을 완화하면서 미백·장벽 강화 시너지

## 주의사항
- 자외선 민감도를 높이므로 반드시 SPF 50+ 자외선 차단제와 함께 사용
- 야간 사용 권장. 아침 사용 시 광감작 위험 증가
- 임신·수유 중 사용 금지 (레티노이드 계열 공통)
- AHA/BHA와 동시 사용 시 과도한 자극 가능. 다른 날 교대 사용 권장
- 처음 사용 시 2~4주간 레티놀 반응(레티노이제이션)이 나타날 수 있으며 이는 정상
