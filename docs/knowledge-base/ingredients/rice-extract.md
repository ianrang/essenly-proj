# 쌀추출물 (Rice Extract)

> 이 정보는 일반 참고용이며 의료 조언이 아닙니다. 피부 상태에 따라 전문의와 상담하세요.

## 기본 정보
- INCI: Oryza Sativa (Rice) Extract / Oryza Sativa (Rice) Bran Extract
- 별명: 쌀겨추출물, 미강추출물, 라이스 워터
- 기능: 미백(페룰산·감마오리자놀 함유), 보습, 항산화, 피부결 개선, 탄력 강화
- K-뷰티에서: 한국 전통 미용법(쌀뜨물 세안)에서 유래한 K-뷰티 대표 한방 성분. 더페이스샵 라이스 세라마이드 라인, 아이소이 쌀 세럼 등이 유명. 쌀 발효 추출물(사케 추출물)은 더 높은 미백 효과

## 피부타입별 적합도
- 건성(dry): 적합 — 쌀겨 오일의 보습 효과와 피부 연화 작용
- 지성(oily): 보통 — 미백·피부결 개선에 효과적이나 오일 성분이 함유된 제품은 무거울 수 있음
- 복합성(combination): 적합 — 전체적인 피부 톤 균일화와 피부결 개선
- 민감성(sensitive): 적합 — 순한 성분으로 민감 피부에서도 안전하게 사용 가능
- 보통(normal): 적합 — 꾸준한 사용으로 맑고 윤기 있는 피부 톤 유지

## 시너지 성분
- 나이아신아마이드: 쌀 추출물의 자연 미백 + 나이아신아마이드의 멜라닌 전달 억제로 미백 시너지
- 세라마이드: 쌀겨에 포함된 천연 지질과 세라마이드가 장벽 강화 시너지

## 주의사항
- 쌀 알레르기가 있는 경우 사용 전 패치 테스트 필요
- 발효 쌀 추출물은 일반 쌀 추출물보다 더 효과적이나, 발효 성분에 민감한 피부는 주의
- 미백 효과를 위해서는 자외선 차단제 병행 필수
