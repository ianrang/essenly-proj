# 살리실산 (Salicylic Acid)

> 이 정보는 일반 참고용이며 의료 조언이 아닙니다. 피부 상태에 따라 전문의와 상담하세요.

## 기본 정보
- INCI: Salicylic Acid
- 별명: BHA (Beta Hydroxy Acid), 베타하이드록시산
- 기능: 모공 속 피지 용해, 각질 제거, 항염·항균, 블랙헤드·화이트헤드 제거
- K-뷰티에서: COSRX BHA 블랙헤드 파워 리퀴드, 이니스프리 BHA 클렌저 등이 대표적. K-뷰티에서는 0.5~2% 저농도로 매일 사용하는 순한 BHA 제품이 인기

## 피부타입별 적합도
- 건성(dry): 주의 — 유분을 제거하여 건조감을 악화시킬 수 있음. 주 1~2회 부분 사용 권장
- 지성(oily): 적합 — 지용성이라 모공 속 피지까지 침투하여 효과적으로 관리
- 복합성(combination): 적합 — T존 위주로 사용하면 효과적
- 민감성(sensitive): 주의 — 낮은 농도(0.5%)로 시작하고 패치 테스트 필수. 항염 효과가 있으나 자극 가능
- 보통(normal): 적합 — 주 2~3회 사용으로 모공 관리와 피부결 개선

## 시너지 성분
- 나이아신아마이드: BHA로 각질 제거 후 나이아신아마이드가 모공 축소와 피부 톤 정리
- 티트리 오일: 항균 효과가 더해져 여드름성 피부에 효과적

## 주의사항
- 한국 화장품법 기준 0.5% 이하만 허용 (전문의약품 제외)
- AHA와 동시 사용 시 과각질 제거 위험. 교대 사용 권장
- 레티놀과 같은 날 사용 시 자극이 심해질 수 있으므로 격일 사용 권장
- 아스피린 알레르기가 있는 경우 주의 필요 (살리실산은 아스피린의 전구체)
