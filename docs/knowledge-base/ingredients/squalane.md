# 스쿠알란 (Squalane)

> 이 정보는 일반 참고용이며 의료 조언이 아닙니다. 피부 상태에 따라 전문의와 상담하세요.

## 기본 정보
- INCI: Squalane
- 별명: 수소화스쿠알렌, 식물성 스쿠알란
- 기능: 에몰리언트(피부 연화), 보습막 형성, 수분 증발 방지, 피부 장벽 강화, 항산화 보조
- K-뷰티에서: 비플레인, 이니스프리, 아로마티카 등에서 순수 스쿠알란 오일 출시. K-뷰티 오일 스킨케어 트렌드의 핵심 성분으로, 올리브·사탕수수 유래 식물성 스쿠알란이 주로 사용됨

## 피부타입별 적합도
- 건성(dry): 적합 — 가벼우면서도 효과적인 보습막을 형성하여 수분 손실 방지
- 지성(oily): 적합 — 인체 피지와 유사한 구조로 무겁지 않고 논코메도제닉(모공 비폐색성)
- 복합성(combination): 적합 — 가벼운 오일감으로 부위별 부담 없이 사용
- 민감성(sensitive): 적합 — 인체 피부에 자연 존재하는 성분(스쿠알렌의 안정화 형태)으로 자극이 거의 없음
- 보통(normal): 적합 — 모든 피부 타입에 적합한 만능 보습 오일

## 시너지 성분
- 히알루론산: HA로 수분 공급 후 스쿠알란으로 밀봉하여 보습 지속력 극대화
- 세라마이드: 지질 성분끼리 조합하여 피부 장벽을 강력하게 보호

## 주의사항
- 스쿠알렌(Squalene)과 다름. 스쿠알란은 수소화 처리하여 산화에 안정적
- 식물성(올리브, 사탕수수 유래)과 동물성(상어 간 유래)이 있으며 K-뷰티에서는 식물성 사용이 대세
- 매우 안전한 성분으로 특별한 부작용 보고가 거의 없음
- 레티놀 등 자극성 성분 사용 후 보호막 역할로 활용 가능
