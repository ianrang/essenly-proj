# 토코페롤 (Tocopherol)

> 이 정보는 일반 참고용이며 의료 조언이 아닙니다. 피부 상태에 따라 전문의와 상담하세요.

## 기본 정보
- INCI: Tocopherol / Tocopheryl Acetate
- 별명: 비타민 E, 알파토코페롤
- 기능: 항산화(지용성 활성산소 중화), 보습, 피부 장벽 보호, 자외선 손상 완화, 상처 치유 보조
- K-뷰티에서: 단독 성분보다는 세럼, 크림, 오일에 항산화 보조 성분으로 폭넓게 활용. 비타민C+E 조합 세럼이 인기이며, 클레어스 등에서 비타민E 함유 오일 제품 출시

## 피부타입별 적합도
- 건성(dry): 적합 — 지용성 성분으로 건조 피부에 보습막을 형성하여 수분 증발 방지
- 지성(oily): 보통 — 오일 기반 제품은 무거울 수 있으나, 세럼에 소량 함유된 것은 무방
- 복합성(combination): 보통 — 건조 부위에 집중 사용하거나 소량 함유 제품 선택
- 민감성(sensitive): 적합 — 자극이 매우 낮고 피부 보호 효과가 있어 민감 피부에 안전
- 보통(normal): 적합 — 일상적 항산화 보호와 피부 건강 유지

## 시너지 성분
- 비타민C(아스코르브산): 비타민C+E 조합은 항산화 효과를 상승시키는 대표적 시너지
- 페룰산: C+E+페룰산 트리오는 피부과학에서 검증된 항산화 골든 조합

## 주의사항
- 고농도 비타민E 오일은 모공을 막을 수 있으므로 여드름성 피부는 주의
- 토코페릴아세테이트(안정화 형태)는 피부에서 토코페롤로 전환되어 작용
- 개봉 후 산화될 수 있으므로 밀폐 보관하고 변색 시 사용 중단
