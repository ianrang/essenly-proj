# 피코 레이저 (Pico Laser)

> 이 정보는 일반 참고용이며 의료 조언이 아닙니다. 시술 전 반드시 전문의와 상담하세요.

## 기본 정보
- 카테고리: laser
- 시술 원리: 피코초(1조분의 1초) 단위의 극초단파 레이저로 색소 입자를 미세하게 분쇄합니다. 기존 나노초 레이저보다 더 작은 입자로 분해하여 체내 흡수를 촉진하고 주변 조직 손상을 최소화합니다.
- 소요 시간: 15~30분
- 다운타임: 1~3일

## 적합 피부타입
- 건성(dry): 보통 - 시술 후 건조감이 있을 수 있으나 보습으로 관리 가능
- 지성(oily): 적합 - 피지 분비와 무관하게 효과적
- 복합성(combination): 적합 - 부위별 차등 없이 균일한 효과
- 민감성(sensitive): 보통 - 기존 레이저 대비 자극이 적으나 테스트 조사 권장
- 보통(normal): 적합 - 최적의 시술 결과 기대 가능

## 적합 피부 고민
- dark_spots: 기미, 주근깨, 검버섯 등 다양한 색소 질환에 효과적
- uneven_tone: 피부 톤 균일화에 탁월
- acne: 여드름 흉터(붉은 자국, 색소 침착) 개선
- sun_damage: 광노화로 인한 색소 침착 개선
- dullness: 콜라겐 리모델링으로 피부 광채 회복

## 시술 횟수 및 비용
- 권장 횟수: 3~5회 (3~4주 간격)
- 가격 범위: ₩100,000 ~ ₩300,000 (1회 기준)

## 시술 전 주의사항
- 시술 2주 전부터 레티노이드, 고농도 비타민C 세럼 사용 중단
- 시술 전 4주 이내 과도한 태닝 금지
- 임신 중이거나 켈로이드 체질인 경우 시술 불가

## 시술 후 관리
- 시술 직후 약간의 붉은기와 부기가 있을 수 있으며 1~3일 내 자연 소멸
- 자외선 차단제(SPF 50+)를 2주 이상 철저히 사용
- 시술 후 1주일간 음주, 사우나, 격렬한 운동 자제
- 한국 피부과에서 최신 장비(피코슈어, 피코웨이 등)를 사용하여 외국인 관광객에게 높은 인기
